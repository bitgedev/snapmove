# Snapmove PRD

---

## 1. 문제 정의

운동을 꾸준히 하는 사람도 자신의 기록을 체계적으로 관리하고 공유할 수 있는 도구가 부족하다. Snapmove는 루틴 관리부터 인증샷 카드 생성까지 운동 기록의 전 과정을 하나의 앱에서 해결한다.

---

## 2. 타겟 사용자

운동 루틴을 관리하고 싶지만 기록이 흩어져 있고, SNS에 운동 인증을 올리고 싶지만 직접 편집하기 번거로운 20~30대 운동 입문자 및 중급자.

---

## 3. 제안 해결책

루틴 생성부터 운동 기록, 감성적인 인증샷 카드 자동 생성 및 저장까지 하나의 앱에서 처리할 수 있는 올인원 운동 트래커를 제공한다.

---

## 4. 목표 및 성공 지표

운동 기록과 공유 경험을 단순화하여 사용자가 꾸준히 앱을 사용하게 만드는 것을 목표로 한다. 성공 지표는 루틴 완료율, 인증샷 카드 저장/공유 횟수, 재방문율로 측정한다.

---

## 5. 핵심 기능 (MVP)

**인증 (Auth)**

- 이메일 회원가입 / 로그인
- Supabase Auth 기반 세션 관리
- 로그인 상태에 따른 라우팅 보호

**루틴 관리**

- 루틴 생성 / 수정 / 삭제
- 루틴 내 운동 추가 / 삭제
- 운동별 세트 / 횟수 / 무게 설정

**운동 기록 / 트래킹**

- 루틴 실행 후 세트별 실제 수행 기록
- 날짜별 운동 히스토리 조회

**인증샷 카드**

- 운동 완료 시 카드 자동 생성
- 카드 내 날짜 / 운동 목록 / 총 볼륨 표시
- 감성적인 디자인으로 이미지 저장 (PNG)
- 인스타그램 비율 대응 (1:1, 4:5)

---

## 6. 기술 스택

- Frontend: Next.js / TypeScript / Tailwind CSS
- Backend: Supabase (Auth, Database, Storage)
- 인증샷 카드: html2canvas
- 배포: Vercel

---

## 7. 페이지 구조

| 페이지    | 경로             | 설명                                       |
| --------- | ---------------- | ------------------------------------------ |
| 랜딩      | `/`              | 서비스 소개 및 로그인 유도                 |
| 로그인    | `/login`         | 이메일 로그인                              |
| 회원가입  | `/signup`        | 이메일 회원가입                            |
| 루틴 목록 | `/routines`      | 내 루틴 목록                               |
| 루틴 상세 | `/routines/[id]` | 루틴 내 운동 목록                          |
| 운동 실행 | `/workout/[id]`  | 운동 기록 입력 및 완료 시 인증샷 카드 생성 |
| 히스토리  | `/history`       | 날짜별 운동 기록                           |

---

## 8. 확장 가능 기능 (Post-MVP)

- 무게 증가 그래프 (데이터 시각화)
- 운동 통계 대시보드
- 스트릭 히트맵
- 소셜 피드 / 친구 기능
